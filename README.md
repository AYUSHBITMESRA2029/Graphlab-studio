# GraphLab Studio

GraphLab Studio is a standalone browser app for science stream students. It can plot:

- mathematics equation graphs
- parametric curves
- vector representation graphs
- electrical engineering phasor diagrams
- scatter plots with trend lines
- bar charts
- saved graph history in the browser
- animated value cursor on plotted graph paths and phasor/vector traces

Open `index.html` in a browser to run it. No install step is required.

Use **Plot graph** or **Interpret request** to save a graph into history. Click a history item to restore its stream, graph type, inputs, and plotted view.

## Input formats

- Equation: `sin(x) + 0.35*x`
- Parametric: `x(t)=4*cos(t)`, `y(t)=2*sin(t)`
- Vector row: `Force A,4,3,#1f9bd1`
- Phasor row: `Voltage,230,0,#ef4444`
- Scatter row: `2,3.5`
- Bar row: `Algebra,86,#1f9bd1`
